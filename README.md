# spaghetticode
Count the number of rows in source files in a directory

Example:

spaghetticode . cs cshtml sql

will count all rows in files .cs, .cshtml and .sql 

```
|====================|===============|
| Extension          | Rows          |
|====================|===============|
| .cs                | 197760        |
| .cshtml            | 48270         |
| .sql               | 42402         |
|====================|===============|
| Total              | 288432        |
|====================|===============|
```
